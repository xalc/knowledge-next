### 分类变量

TP FP  FN TN

精度（accuracy）= (TP + TN) /(TP + FP + FN + TN)

错误率( errorrate) = (FP + FN) /(TP + TN + FP + FN)

查准率（精确率）(Precision) = (TP)/(TP + FP)

查全率(招回率)(Recall) = (TP)/(TP + FN)
